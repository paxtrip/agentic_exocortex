# Unified RAG System

## Что это такое?

Это умная система поиска по вашим личным заметкам в SiYuan. Она может:

- **Отвечать на вопросы** на основе ваших заметок
- **Искать код** из ваших записей
- **Показывать связи** между идеями в ваших текстах
- **Работать полностью оффлайн** - ваши данные не уходят в интернет
- **Понимать русский язык** так же хорошо, как английский

Система поддерживает от 1000 до 10000 документов и работает на обычном VPS сервере.

## Как это работает?

### Три уровня работы (graceful degradation)

1. **Полный ответ**: Система использует ИИ (Gemini, Groq) для подробных ответов
2. **Умный поиск**: Если ИИ недоступен, система находит точные цитаты из ваших заметок
3. **Простой поиск**: В крайнем случае работает как обычный поиск по тексту

### Что внутри?

- **API сервер** на FastAPI - обрабатывает запросы
- **Плагин для SiYuan** - интеграция с вашим блокнотом
- **Qdrant** - быстрая векторная база данных для поиска
- **SQLite** - база данных для хранения ваших данных
- **Traefik** - маршрутизатор для доменов

## Быстрый запуск

### Что нужно?

- Docker и Docker Compose (установите на ваш сервер)
- VPS сервер с Ubuntu 24.04, 4 ядрами CPU и 8GB RAM
- Домены: `api.вашдомен.com`, `qd.вашдомен.com`, `siyuan.вашдомен.com`

### Шаг 1: Скачайте код

```bash
git clone <repository-url>
cd agentic_exocortex
```

### Шаг 2: Настройте переменные

```bash
cp .env.example .env
```

Откройте `.env` файл и заполните:
- `DOMAIN_NAME=вашдомен.com`
- `SIYUAN_AUTH_CODE=код_доступа_из_siyuan`

### Шаг 3: Запустите систему

```bash
docker compose -f infrastructure/docker-compose.yml up -d
```

### Шаг 4: Проверьте работу

```bash
curl https://api.вашдомен.com/health/ready
```

Должны получить ответ вроде:
```json
{"status": "healthy", "uptime_seconds": 123.45}
```

## Как использовать?

### Через API

#### Спросить что-то и получить ответ

```bash
curl -X POST https://api.вашдомен.com/ask \
  -H "Content-Type: application/json" \
  -d '{
    "query": "Как настроить векторный поиск?",
    "context_limit": 5,
    "generate_answer": true
  }'
```

Получите ответ вроде:
```json
{
  "answer": "Для настройки векторного поиска нужно...",
  "sources": [
    {
      "title": "Мои заметки о базах данных",
      "url": "siyuan://blocks/20231201120000-abc123",
      "confidence": 0.87,
      "citations": ["стр. 15-17"]
    }
  ],
  "processing_time_ms": 1250
}
```

#### Просто найти информацию

```bash
curl "https://api.вашдомен.com/search?q=векторный+поиск&limit=10"
```

Получите список найденных документов:
```json
{
  "total_results": 5,
  "results": [
    {
      "title": "Векторные базы данных",
      "content_preview": "Векторный поиск основан на...",
      "confidence": 0.92,
      "url": "siyuan://blocks/20231201120000-abc123"
    }
  ]
}
```

### Через SiYuan плагин

1. Установите плагин в SiYuan
2. Откройте панель поиска
3. Введите вопрос или код для поиска
4. Получите результаты прямо в интерфейсе SiYuan

#### GET `/health/*`
Эндпоинты мониторинга здоровья системы.

### Ошибки

- `400 Bad Request`: Некорректные параметры запроса
- `503 Service Unavailable`: Сервис временно недоступен
- `500 Internal Server Error`: Внутренняя ошибка сервера

## Настройка

### Основные настройки

В файле `.env` нужно указать:
