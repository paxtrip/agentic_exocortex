# Руководство пользователя: Ваша умная система поиска по заметкам

Здравствуйте! Это руководство поможет вам установить и использовать вашу личную поисковую систему для заметок SiYuan. Представьте, что у вас есть личный помощник, который прочитал все ваши заметки и может мгновенно ответить на любой вопрос по ним. Эта система — и есть такой помощник.

## Что это такое?

Это программа, которая «читает» ваши заметки из SiYuan, запоминает их и помогает вам быстро находить нужную информацию. Ваши данные при этом остаются только у вас и не передаются в интернет.

## Часть 1: Установка

Установка может показаться сложной, но если следовать шагам, все получится. Весь процесс нужно будет проделать только один раз. Для этого потребуется помощь технического специалиста или хостинг-провайдера, так как программа устанавливается на удаленный сервер.
### Где хранятся мои заметки?

Все ваши заметки и данные для поиска надежно хранятся **только на вашем сервере**. Они не покидают его и не передаются третьим лицам.

- **Оригиналы заметок** находятся в рабочем пространстве SiYuan.
- **Данные для поиска** (индексы и база данных) хранятся рядом, также на вашем сервере.

Это гарантирует полную конфиденциальность и контроль над вашей информацией.


### Что понадобится для установки?

Вашему техническому специалисту потребуется:

1.  **Сервер**: Это как удаленный компьютер, который всегда включен. Подойдет самый простой вариант (VPS) с операционной системой Ubuntu 24.04.
2.  **Доменное имя**: Это уникальный адрес в интернете, например, `vash-pomoshnik.com`.
3.  **Программа Docker**: Специалист установит ее на сервер. Она нужна, чтобы наша система работала.

### Шаги установки (инструкция для специалиста)

Передайте эту часть вашему техническому специалисту.

> 1.  **Клонируйте репозиторий:**
>     ```bash
>     git clone <URL репозитория>
>     cd agentic_exocortex
>     ```
> 2.  **Настройте окружение:**
>     Скопируйте файл с примером настроек:
>     ```bash
>     cp .env.example .env
>     ```
>     Откройте файл `.env` и укажите ваш домен:
>     `DOMAIN_NAME=vash-pomoshnik.com`
> 3.  **Первый запуск для получения токена:**
>     - Запустите только SiYuan, чтобы получить доступ к его интерфейсу:
>       ```bash
>       docker compose -f infrastructure/docker-compose.yml up -d siyuan
>       ```
>     - Подождите около минуты, пока сервис запустится.
>     - Откройте в браузере `http://IP-адрес-вашего-сервера:6806`.
>     - Пройдите первоначальную настройку SiYuan.
>     - Зайдите в «Настройки» -> «О программе».
>     - Найдите и скопируйте «API токен».
>
> 4.  **Добавьте токен в настройки:**
>     - Теперь остановите временный запуск:
>       ```bash
>       docker compose -f infrastructure/docker-compose.yml down
>       ```
>     - Откройте файл `.env` и вставьте скопированный токен:
>       `SIYUAN_ACCESS_AUTH_CODE=сюда_вставить_ваш_токен`
>
> 5.  **Полный запуск системы:**
>     Теперь, когда все настройки готовы, запустите все компоненты системы:
>     ```bash
>     docker compose -f infrastructure/docker-compose.yml up -d
>     ```
>     Система начнет свою работу в фоновом режиме.

После этого установка завершена. Теперь можно переходить к наполнению базы.

## Часть 2: Наполнение базы данных (Индексация)

Чтобы система могла отвечать на ваши вопросы, ей нужно «прочитать» все ваши заметки. Этот процесс называется **индексацией**.

### Как запустить индексацию?

1.  Убедитесь, что SiYuan запущен на вашем компьютере.
2.  Откройте плагин системы в SiYuan.
3.  Нажмите кнопку **«Начать индексацию»**.

Процесс может занять некоторое время, если у вас много заметок (от нескольких минут до часа). Система будет в фоновом режиме загружать и обрабатывать ваши тексты. Вы можете продолжать пользоваться заметками как обычно.

После завершения индексации ваша поисковая система готова к работе!

## Часть 3: Как пользоваться системой

Вы можете взаимодействовать с системой двумя способами:

### 1. Через плагин в SiYuan (самый удобный способ)

Это основной и самый простой способ использования.

1.  **Откройте плагин** в боковой панели SiYuan.
2.  Вы увидите **поисковую строку**.
3.  **Задайте вопрос**: напишите свой вопрос простым языком. Например: «Какие книги по психологии я читал в прошлом году?» или «Рецепт яблочного пирога».
4.  **Получите ответ**: Система проанализирует ваши заметки и даст прямой ответ, а также покажет ссылки на заметки, где была найдена информация.

Вы также можете искать не только вопросы, но и фрагменты кода или ключевые слова.
### Как добавить свои документы для поиска?

Система работает со всеми заметками, которые находятся **внутри вашего SiYuan**. Если вы хотите добавить для поиска внешнюю документацию (например, папку с `.md` файлами), вам нужно просто **импортировать ее в SiYuan**.

1.  Откройте SiYuan.
2.  Перетащите вашу папку с файлами прямо в окно SiYuan или используйте меню "Импорт".
3.  После того как файлы появятся в SiYuan, запустите **индексацию** в плагине.

Система обработает новые документы, и вы сможете выполнять по ним поиск.


### 2. Через API (для продвинутых пользователей)

Если вы хотите интегрировать поиск с другими программами, вы можете использовать API. Это специальный способ для программ общаться друг с другом.

**Пример: как задать вопрос через API**

Откройте терминал (командную строку) и выполните команду:

```bash
curl -X POST https://api.vash-pomoshnik.com/ask \
  -H "Content-Type: application/json" \
  -d '{
    "query": "Как настроить векторный поиск?",
    "generate_answer": true
  }'
```

В ответ вы получите структурированный текст с ответом и источниками.

## Часть 4: Управление языковыми моделями (LLM)

Ваша система может использовать искусственный интеллект (ИИ) для генерации ответов на вопросы. Вы можете выбирать, какие именно ИИ-модели использовать, или не использовать их вовсе.

### Как это работает?

Система автоматически выбирает лучшую доступную модель по следующему приоритету:

1.  **Gemini** (от Google) - высокое качество ответов.
2.  **Groq** - очень высокая скорость генерации.
3.  **OpenRouter** - доступ к разным моделям, включая бесплатные.

Чтобы **активировать** модель, вам нужно получить её **API-ключ** (специальный код доступа) на сайте провайдера и добавить его в файл настроек `.env`.

### Как настроить?

1.  **Выберите провайдера**: решите, какую модель вы хотите использовать (например, Gemini).
2.  **Получите API-ключ**: зарегистрируйтесь на сайте провайдера (например, [Google AI Studio](https://aistudio.google.com/) для Gemini) и создайте ключ.
3.  **Добавьте ключ в файл `.env`**:
    - Откройте файл `.env`.
    - Найдите строку для нужного провайдера и вставьте ваш ключ. Например:
      ```
      GEMINI_API_KEY=сюда_вставить_ваш_ключ_от_gemini
      ```
    - Вы можете добавить ключи для нескольких провайдеров. Система сама выберет лучший из доступных.

### Что если не добавлять ключи?

Если вы не добавите ни одного API-ключа, система **все равно будет работать**. Вместо генерации новых ответов она будет выполнять **умный поиск** и показывать вам точные цитаты из ваших заметок, которые соответствуют вашему запросу. Ваши данные при этом, как и всегда, остаются в полной безопасности на вашем сервере.

      ```
      GEMINI_API_KEY=сюда_вставить_ваш_ключ_от_gemini
      GROQ_API_KEY=сюда_вставить_ваш_ключ_от_groq
      OPENROUTER_API_KEY=сюда_вставить_ваш_ключ_от_openrouter
      ```
    - Вы можете добавить ключи для нескольких провайдеров. Система сама выберет лучший из доступных.

#### Выбор модели для OpenRouter

Если вы используете OpenRouter, вы можете указать, какую именно модель использовать, добавив в файл `.env` переменную `OPENROUTER_MODEL`:

```
OPENROUTER_MODEL=anthropic/claude-3-haiku
```

Полный список доступных моделей можно найти на сайте OpenRouter. Если эту переменную не указывать, будет использоваться бесплатная модель по умолчанию.
